# Kripto Shop – MERN E-commerce App

## 📦 Tech Stack

- MongoDB + Mongoose
- Express.js
- React.js + Vite
- Node.js
- Stripe for Payments
- Tailwind CSS

---

### Clone the Repo

```bash

cd kripto-shop








Backend Setup



cd backend
npm install

# Fill your MongoDB URI and Stripe keys
npm run dev





Frontend Setup


cd frontend
